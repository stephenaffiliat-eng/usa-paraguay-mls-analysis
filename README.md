# USA vs Paraguay — MLS Player Impact Analysis

This repository contains the full analytical workflow behind my Substack article.

## 🎯 Focus of the analysis
Three MLS-system players:
- Matthew Freese  
- Tim Ream  
- Sebastian Berhalter  

## 📊 What’s inside
- `usa_paraguay_analysis.ipynb` — full Python notebook  
- `data/usa_paraguay_players.csv` — match dataset  
- `visuals/` — heatmaps, passing charts, defensive impact  

## 📈 Metrics analyzed
- Passing accuracy  
- Opposition-half passing  
- Defensive actions  
- Recoveries  
- Duels  
- Long balls  
- Possession lost  
- xG / xA  
- Shots
